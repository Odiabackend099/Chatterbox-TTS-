STEP 1: IMPORT WORKFLOW (1 minute)
───────────────────────────────────────────────────────────────────────

1. Open n8n in your browser
2. Click "Workflows" → "Import from File"
3. Select: n8n/COMPLETE_TTS_WORKFLOW.json
4. Click "Import"

✅ You should see 4 connected nodes


STEP 2: ACTIVATE (10 seconds)
───────────────────────────────────────────────────────────────────────

1. Find toggle switch at top-right
2. Turn it ON (green)

✅ Switch should say "Active"


STEP 3: TEST IT (1 minute)
───────────────────────────────────────────────────────────────────────

1. Click "Webhook Trigger" node
2. Copy the "Production URL"
3. Run in terminal:

   cd /Users/odiadev/chatterbox-twilio-integration
   ./quick_n8n_test.sh "YOUR_WEBHOOK_URL_HERE"

✅ Should see "🎉 ALL TESTS PASSED!"

❌ Connection Refused
   → n8n not running
   → Wrong URL/port
